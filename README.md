## NBA Stats Compiler

A web app for searching and viewing NBA player stats from the 2023-24 season.

🔗 **Live Demo:** [https://nba-stats-compiler.vercel.app](https://nba-stats-compiler.vercel.app)

## Tech Stack

- **Frontend/Backend:** Next.js (App Router)
- **Database:** Supabase (PostgreSQL)
- **Deployment:** Vercel

## Features

- Search for any NBA player by name
- View full 2023-24 season stats
